# Recorded interface audio

`percussion/hi-hat-close.wav` is derived from **Hi-hat close #5**, recorded by Joseph Sardin and
published by BigSoundBank under CC0. Chord Hero shortened the recording to a 220 ms one-shot,
added a short fade, and converted it to mono 44.1 kHz 16-bit PCM WAV.
