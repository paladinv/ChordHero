# Guitar samples

Drop real recorded mono or stereo `.wav` files here to enable sample playback:

- `steel-strum.wav`
- `steel-arpeggio.wav`
- `nylon-strum.wav`
- `nylon-arpeggio.wav`
- `muted-muted-strum.wav`
- `picked-arpeggio.wav`

The Chord Library attempts to load these files first and falls back to generated Web Audio previews if a file is missing.
